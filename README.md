# CODE-COMPILER

Frontend application for the Code Compiler project built with React + Vite.

## Features

- **React** with Vite for fast development
- **HMR** (Hot Module Replacement) for instant updates
- **ESLint** configuration for code quality
- **Code Editor** component
- **Terminal** component

## Tech Stack

- React
- Vite
- Axios (for API calls)
- ESLint
